Plan title: Add equal-opportunity fairness penalty to BCE loss

Applied edit (file: train.py, symbol: train_one_epoch)
- After `logits = model(features)` and the BCE `loss = criterion(logits, labels)`
  (ANCHOR line preserved), inserted a soft per-group TPR variance penalty:
    probs = torch.sigmoid(logits)
    labels_f = labels.unsqueeze(-1)
    group_onehot = features[:, -5:]   # ETH block is last 5 columns of features
    pos_per_group = (labels_f * group_onehot).sum(dim=0)
    tp_per_group  = (probs.unsqueeze(-1) * labels_f * group_onehot).sum(dim=0)
    present = pos_per_group >= 1.0
    tpr_g = tp_per_group / pos_per_group.clamp_min(1.0)
    if present.any():
        fairness_loss = tpr_g[present].var(unbiased=False)
        loss = loss + 0.05 * fairness_loss

- The `loss` variable that is backpropped is BCE + 0.05 * fairness_loss.
